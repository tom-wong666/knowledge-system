
# bash

cd // 进入某个文件  
cd ~ // 进入根目录  
cd .. // 进入上一级  
左键复制 右键黏贴  
mkdir [-p] 目录名 // 创建新文件  
说明:  
参数：-p 确保目录名称存在，如果目录不存在的就新创建一个  

$ rm -rf 文件夹名字 // 删除某个文件夹  
说明：  
-r 向下递归，不管有多少级目录，一并删除  
-f 直接强行删除，不作任何提示的意思  
sudo是linux系统管理指令，是允许系统管理员让普通用户执行一些或者全部的root命令的一个工具，如halt，reboot，su等等  
如果需要管理员权限，则在前面增肌sudo即sudo rm -rf 文件夹的名字  

vim 文件名 // 打开某个文件的编辑界面
insert // 键位，进入输入状态
esc q! // 不保存退出
esc wq // 保存退出
